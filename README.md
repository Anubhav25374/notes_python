# 🐍 Core Python Practice Repository

This repository contains programs and small projects created while learning and practicing **Core Python concepts**.  
The main goal of this repository is to strengthen **programming fundamentals**, improve **problem-solving skills**, and maintain a structured collection of Python learning exercises.

Python is one of the most widely used programming languages in fields such as **software development, data science, machine learning, automation, and web development**.  
This repository focuses on building a strong foundation in Python before moving on to advanced topics.

---

# 📚 Topics Covered

This repository includes examples and exercises covering the following Python fundamentals:

### 1. Python Basics
- Variables and data types
- Type conversion
- Input and output operations
- Comments and code structure

### 2. Operators
- Arithmetic operators
- Comparison operators
- Logical operators
- Assignment operators

### 3. Control Flow
- Conditional statements (`if`, `elif`, `else`)
- Nested conditions
- Ternary operators

### 4. Loops
- `for` loops
- `while` loops
- Loop control statements
  - `break`
  - `continue`
  - `pass`

### 5. Functions
- Function definition
- Parameters and arguments
- Return values
- Lambda functions
- Recursion basics

### 6. Data Structures
- Lists
- Tuples
- Dictionaries
- Sets
- Basic operations and methods

### 7. File Handling
- Reading files
- Writing files
- Appending data
- Working with text files

### 8. JSON Handling
- Reading JSON files
- Writing JSON data
- Converting Python objects to JSON

### 9. Exception Handling
- `try`
- `except`
- `finally`
- Custom error handling

### 10. Mini Projects
Small command-line based projects to apply learned concepts.

Example:
- YouTube video manager
- File-based data storage
- Simple data management scripts

---


Each folder focuses on a specific Python concept to keep the repository **organized and easy to navigate**.
